# Kroniki Marty - Interactive Wellness Platform

## Project Overview
A modern Single Page Application (SPA) designed to provide an immersive user experience through advanced WebGL technologies.

## Key Technical Features
- **3D Interactive UI:** Developed a dynamic "Quantum Field" background using **Three.js** and **WebGL**, responding to mouse movements and touch gestures.
- **Performance Engine:** Implemented a hardware-aware particle system (switching between 4k and 8k particles) to maintain a constant **60 FPS** on both mobile and desktop.
- **Automated Lead System:** Custom integration with **Telegram API** via PHP for instantaneous notifications and lead management.
- **Advanced Animations:** Utilized the **GSAP** library for high-performance entry animations and smooth transitions.

## Tech Stack
- **Frontend:** JavaScript (ES6+), Three.js, GSAP, CSS Grid/Flexbox.
- **Backend:** PHP, Telegram Bot API.
- **Design:** Glassmorphism, Responsive Web Design.
